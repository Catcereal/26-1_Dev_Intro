# 개발 입문 스터디 1주차

VSCODE(편집기), Git, Github(플랫폼), Vercel(배포 플랫폼) 등 개발의 기본적인 환경을 구성했다

가장 중요한 것

```
git status
git add <file>
git commit -m "<내용>"
git push
```

Git 은 Version Control을 위해서 사용하는 프로그램이다.
앞으로도 계속 우리는 Git을 이용해서 스터디를 진행할 예정이다.

index.html 페이지를 수정해서, 위의 4개로 커밋 하고 푸쉬해줬음
이후 Vercel로 배포했다
Vercel을 통해서 Repository를 Import 해줘서, index.html이 웹페이지로 Deploy가 되었다.

배포한 Vercel URL : https://26-1-dev-intro-lilac.vercel.app/
