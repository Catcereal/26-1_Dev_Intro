## 새로 배우는 명령어

git clone <URL>
cd .. / cd <파일명>
ls
pwd

- **pwd** : 현재 폴더(directory) 위치 /[Print Working Directory/]
- **git clone <URL>** : github 레포지토리 코드를 컴퓨터로 가져오기
- **ls** : 지금 위치에 어떤 파일이 있는지 보기
- **cd 명령어**
  - cd <폴더명> : Change Directory -> 지금 위치에서 폴더 위치를 변경하기
  - cd .. : 폴더 밖으로
  - cd / : 루트로 돌아가기

## CSS 알아보기

**Cascading Style Sheets**
HTML 등의 마크업 언어로 작성된 문서가 실제로 표현되는 방법을 정해주는 스타일 시트 언어  
쉽게 말하면 HTML에 스타일을 표현하게 해주는 언어

### HTML의 한계

HTML로 만든 문서는 이쁘지 않음... <br>
웹 문서를 만들기 위해서 탄생했기 때문  
이를 위해서 CSS를 사용함

### 인라인 스타일

예시)

```HTML
<h1 style="
    color:yellow;
    background-color:red;
    ">신짱구</h1>
```

**인라인 스타일** : 태그에 직접적으로 스타일을 입히는 방식
